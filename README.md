# 🏏 IPL Data Analysis — EDA Project

## 📌 About
Exploratory Data Analysis on IPL match data
using Python, Pandas, Matplotlib and Seaborn.

## 📊 Questions Answered
- Which team won the most matches?
- Does toss affect match result?
- Who are the top performers?
- Which venues host the most matches?
- What were the closest and biggest wins?

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📁 Dataset
IPL match-level data including teams, venues,
scores, players and match results.
